# Административная панель

Небольшая административная панель для отображения и управления списком пользователей системы

## Реализованный функционал

- отображение списка пользователей в таблице
- сортировка списка по всем колонкам (по возрастанию и убыванию)
- фильтрация списка по всем колонкам
- добавление новых пользователей через модальное окно
- редактирование существующих пользователей через модальное окно
- удаление пользователей с подтверждением

## Особенности реализации
- список пользователей генерируется на основе описанного типа данных с помощью библиотеки faker и сохраняется в localStorage
- таблица реализована через CustomTable (shared компонент), который принимает список колонок и строк
- для создания и редактирования используется UserForm с валидацией обязательных полей и корректности email
- все тексты (названия колонок, кнопок, заголовки диалогов) вынесены в константы в shared/config/texts, чтобы облегчить поддержку и локализацию
- кнопка добавления пользователя расположена вверху таблицы для удобства пользователей, новые пользователи добавляются в начало списка
